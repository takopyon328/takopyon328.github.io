# ニコニコ風コメントスクリーン

授業中に学生がスマホ・PCから送ったコメントを、教員PCの画面に
ニコニコ動画のように右から左へ流して表示するアプリです。

サーバーは Python 標準ライブラリのみで動くため、**追加インストールは不要**です
(Python 3.7 以上がインストールされていればOK)。

## 仕組み

```
学生のスマホ/PC ──(同じWi-Fi)──> 教員PC(server.py) ──> 表示ページにコメントが流れる
   投稿ページ                        ポート8000で待受け        プロジェクターに映す
```

## 使い方

### 1. 教員PCでサーバーを起動する

`server.py` を教員PCに置き、コマンドプロンプト(またはターミナル)で実行します。

```
python server.py
```

起動すると、表示ページのURLと学生に伝えるURLが表示されます。

```
  【表示ページ(このPCで開く)】
    http://localhost:8000/screen

  【投稿ページ(学生に伝えるURL)】
    http://192.168.1.23:8000/      ← IPアドレスは環境により異なります
```

ポートを変えたい場合は `python server.py 8080` のように指定できます。

### 2. 表示ページを開く

教員PCのブラウザで `http://localhost:8000/screen` を開き、
プロジェクターに映します。

- 画面右下に学生用のURLとQRコード(インターネット接続時のみ)が表示されます
- **F キー**: 全画面表示の切り替え
- **H キー**: 右下の案内(URL・QRコード)の表示/非表示
- **C キー**: 流れているコメントを全て消す

スライドの上にコメントだけを重ねたい場合は、OBS等のクロマキー機能と
組み合わせて `http://localhost:8000/screen?bg=green` (緑背景)を使えます。

### 3. 学生がコメントを投稿する

学生は教員PCと**同じネットワーク(Wi-Fi)**に接続し、画面に表示された
URL(例: `http://192.168.1.23:8000/`)にスマホやPCのブラウザでアクセスします。

投稿ページではコメントの **色(7色)** と **大きさ(小・中・大)** を選べます。

## 設定の変更

`server.py` の冒頭にある定数を編集すると挙動を変えられます。

| 定数 | 意味 | 初期値 |
|---|---|---|
| `MAX_COMMENT_LENGTH` | コメントの最大文字数 | 60 |
| `MIN_POST_INTERVAL` | 同じ端末からの連続投稿の最小間隔(秒) | 1.0 |
| `NG_WORDS` | 送信を禁止する単語のリスト | なし |

## うまく動かないとき

- **学生がアクセスできない**
  - 学生の端末が教員PCと同じネットワークに接続されているか確認してください
    (大学の無線LANでは端末間通信が制限されている場合があります。その場合は
    モバイルルーターやスマホのテザリングで専用のネットワークを作ると確実です)
  - Windowsのファイアウォールで初回起動時に「アクセスを許可する」を
    選んでください(許可し忘れた場合は、コントロールパネルの
    「Windows Defender ファイアウォール」から Python を許可します)
- **ポートが使用中というエラーが出る**
  - 別のポート番号で起動してください: `python server.py 8080`
- **QRコードが表示されない**
  - QRコードの生成のみ外部サービスを使っているため、インターネットに
    接続していない教室では表示されません(URLを口頭・板書で伝えてください)
